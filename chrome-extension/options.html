AudioRouter Tabs

Device access

Chrome hides audio device names ("MacBook Pro Speakers", "AirPods", …) until an extension is granted media-device access. Chrome only offers this through the microphone permission — AudioRouter never records or listens to anything; the permission is requested once, released immediately, and used purely to reveal the names of your output devices. The code is open source if you want to verify.

Tips

• To route a tab: open it, click the AudioRouter icon, and pick a device or move the volume slider.
• Routing survives page navigation and stops automatically when the tab closes.
• If the Mac app has a whole-Chrome rule active, it overrides per-tab routing — remove Chrome's rule in the menu bar app when using per-tab control.
